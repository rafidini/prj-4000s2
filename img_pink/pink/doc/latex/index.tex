Pink is an image processing library developed at {\tt E\-S\-I\-E\-E Engineering} for research and teaching purposes. It contains implementations of over 200 algorithms for image segmentation and filtering. Most of the operators come from mathematical morphology, but it contains operators from different fields. It is a free software licensed under the {\tt Ce\-C\-I\-L\-L } license.

In august 2011 Pink has been presented at the European Python Scientific Conference as a poster. Below you can see the conference poster (also in {\tt pdf } and hi-\/res {\tt jpeg }).
